Algebra Generator API

To see an outcome for the API, go to this link. There are five levels to choose from. Change the number at the end of the link to 0-5 to select that level.

The Python server script can be found here.

Explanation:

This project spawned from me wanting to experiment with RESTful API logic. I decided to use my algebra generator as the product of the API. The API URLs can be used to fetch equations from the API as a JSON object.

What It Does:

The user (or a fetch request code) creates the URL with the desired level of the equation as a query. The API then generates a set of equation data using the same code as my algebra generator, before putting in a JSON object and returning it.

Final thoughts:

This project isn't technically complete yet. I would like to eventually create an interface for it. The way the solution is written too works in a console but might need to be edited for browser display. I would also like to experiment with rendering the outcome using some form of LaTeX renderer. I did however, learn exactly what I intended with this project. Namely, how to create a basic API that allows a server to return something from a Python script.
